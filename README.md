# USACO Problems

Solutions and notes from working through USACO problems.

This repo is my depth-practice track for algorithmic thinking — the focus is on fully understanding each solution: correctness, time/space complexity, and edge cases, not just getting an "Accepted" verdict.

## Approach

- Attempt each problem cold first, no hints
- If stuck, look at the topic/pattern only before reading a full solution
- After solving (or reading a solution), re-implement it from memory
- Revisit failed/hinted problems 2–3 days later to check retention
